# Weather-Forecast
🌦️ Weather Forecast Website
A responsive and user-friendly weather forecasting web application that provides real-time weather information for cities around the world. Users can search for any city and instantly view weather details such as temperature, humidity, wind speed, and weather conditions.

🚀 Features
  Search weather by city name
  Real-time temperature updates
  Humidity information
  Wind speed details
  Weather condition display
  Fully responsive design
  Fast and interactive user experience
  Dynamic data fetched using Weather API
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Weather API
Fetch API
